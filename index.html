欢迎学习开发Ruby语言软件包

　　Antetype软件包是为了学习Ruby语言软件包而开发的。Antetype软件包每一个版本就是一个开发软件包实例。

开发和学习Antetype软件包系统环境

Mac OS X系统版本 10.5.8以上
Ruby语言版本 1.9.2以上

体验软件包

$ gem install antetype -v=0.0.3.1
$ antetyepe
$ antetyepe show_antetype_dir
$ antetyepe version # 存在问题是为了以后学习之用。

获取源代码

$ git clone https://github.com/cnruby/learn_gems.git
$ cd learn_gems/antetype003/antetype
$ bundle update
$ antetyepe
$ antetyepe show_antetype_dir
$ antetyepe version # 没有问题。
$ rake -T
$ bundle exec rake cucumber:cucumber # 测试代码
$ open coverage/index.html
# 等等
